## Thème choisi

J'ai choisi le thème pro, afin d'opter pour un style minimaliste et simple.Bien que le style RPG aurait pu permettre de sortir des sentiers battus.

## Lancer le projet

Pour lance le projet, à la racine npm run dev.

## Fonctionnalités implémentées

Les fonctionnalités implementés sont l'ajout,la suppression et la modification de tâches.La possibilité de passer une tâche d'un état à un autre. Implémentation du localStorage pour garder les modifications dans le navigateur. Je me suis aidé d'ia pour certaines parties notamment le style.

**Bonus**
J'ai implementé un bandeau contenant des statistiques sur les tâches et le thème clair et sombre.
